## Joy Plot Panel

Show joy plots in grafana

### Screenshots

![example](https://raw.githubusercontent.com/NatelEnergy/grafana-joy-panel/master/src/img/jd.jpg)

### Building

To complie, run:

```
npm install -g yarn
yarn install --pure-lockfile
grunt
```

To lint everything:

```
yarn pretty
```
